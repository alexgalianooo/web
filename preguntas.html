Preguntas y Respuestas

1.Red Informática:

Una red informática es un conjunto de dispositivos electrónicos interconectados que comparten recursos y datos. Estos dispositivos pueden incluir computadoras, servidores, routers, y otros dispositivos de hardware y software.

2.Red LAN (Local Area Network):

Es una red que se limita a un área geográfica relativamente pequeña, como un edificio o campus. Las LAN permiten la conexión y comunicación de dispositivos dentro de ese espacio limitado.

3.Red WAN (Wide Area Network):

A diferencia de una LAN, una WAN cubre una amplia área geográfica, como una ciudad, un país o incluso a nivel mundial. Las WAN conectan varias LAN y permiten la comunicación a larga distancia.

4.Internet:

Internet es una red global de redes interconectadas que utilizan el protocolo de Internet (IP) para la comunicación. Permite el intercambio de información y servicios a nivel mundial.

5.Web:

La web se refiere a la World Wide Web (WWW), que es una parte de Internet que consiste en páginas web interconectadas mediante hipervínculos. La web facilita la navegación y el acceso a información a través de navegadores web.

6.WWW (World Wide Web):

Es un sistema de información en la web que permite acceder a documentos, imágenes, videos y otros archivos a través de enlaces utilizando navegadores web.

7.Protocolo HTTP y HTTPS:

HTTP (Hypertext Transfer Protocol): Es el protocolo de transferencia de hipertexto utilizado para la transmisión de datos en la World Wide Web. HTTPS (Hypertext Transfer Protocol Secure): Es una versión segura de HTTP que utiliza cifrado SSL/TLS para proteger la información transmitida.

8.Página web estática vs. página web dinámica:

Estática: Contiene contenido fijo que no cambia, a menos que se edite manualmente. Dinámica: Genera contenido de forma dinámica en respuesta a la interacción del usuario o a datos almacenados en una base de datos.

9.CMS (Sistema de Gestión de Contenidos):

Es una aplicación que facilita la creación, edición y gestión de contenido en línea. Ejemplos incluyen WordPress, Joomla y Drupal. Ventajas: facilidad de uso; inconvenientes: posible limitación en personalización.

10.Proveedor de servicios de Internet (ISP):

Empresas que ofrecen acceso a Internet. Ejemplos: Comcast, AT&T, Movistar.

11.Dirección IP:

Identificador numérico asignado a cada dispositivo en una red. Diferencia entre pública (identifica el dispositivo en Internet) y privada (identifica el dispositivo en una red local).

12.Alojamiento web (Hosting):

s el servicio que permite almacenar y acceder a archivos de un sitio web a través de Internet. Ejemplos de proveedores: Bluehost, HostGator, AWS.
